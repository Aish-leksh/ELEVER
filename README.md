# ELEVER - Personal Blog Web Application

A modern and responsive **Personal Blog Web Application** built using **Python** and **Django**. ELEVER provides a clean platform where users can register, log in, read and manage blog posts through an intuitive and user-friendly interface.

---

## Features

- 🔐 User Registration & Login
- 👤 Secure User Authentication
- 📝 Create and Publish Blog Posts
- 📖 Read Blog Posts
- 📂 Blog Categories
- 🏠 Responsive Home Page
- 📊 User Dashboard
- ℹ️ About Page
- 📞 Contact Page
- ⚙️ Django Admin Panel

---

## Tech Stack

- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** SQLite3
- **Version Control:** Git & GitHub

## Installation

### Clone the Repository

```bash
git clone https://github.com/Aish-leksh/ELEVER.git
```

### Navigate to the Project

```bash
cd ELEVER
```

### Create a Virtual Environment

```bash
python -m venv env
```

### Activate the Environment

**Windows**

```bash
env\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
